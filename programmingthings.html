Linear Interpolation

This first code is a simple linear interpolation calculator where you need to make sure you add the right y's and x's!

x = float(input("Enter value of x: "))
y1 = float(input("Enter value of y1: "))
y2 = float(input("Enter value of y2: "))
x1 = float(input("Enter value of x1: "))
x2 = float(input("Enter value of x2: "))

ans = y1 + (x-x1)*(y2-y1)/(x2-x1)

print("Answer is: ", ans)

Conversion Calculator

This is a very basic conversion calculator that you can easily alter to fit the conversions that you need.

conv = input("""A for kWh -> MJ
B for MJ -> kWh: """)

if conv == 'A':
    kwh = float(input("Enter amount of kWh: "))
    mj = 3.6 * kwh
    print("The amount in MJ is: %f MJ" % mj)
else:
    mj = float(input("Enter amount of MJ: "))
    kwh = mj / 3.6
    print("The amount in kWh is: %f kWh" % kwh)
